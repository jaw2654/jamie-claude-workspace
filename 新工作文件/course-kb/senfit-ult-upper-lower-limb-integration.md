# ULT 上下肢整合訓練 2.0

> Jamie 在 SenFit 史博比老師 2 天研習課的筆記。這份是 **代償歸零招牌詞** 的方法論根據。

---

## 原文筆記（教練級 · 完整技術細節）

### 一、核心方法論架構

#### 1.1 教學四步驟
```
發現問題 → 分類問題 → 拆解問題 → 整合訓練
辨識代償   分類提示   拆解評估   對應訓練
```
> 「訓練即是評估」— 每次訓練動作都在蒐集代償情報

#### 1.2 相鄰關節假說（Joint by Joint）
- 活動關節 ↔ 穩定關節 交錯設計
- 從腳到肩膀：腳踝（活）／膝（穩）／髖（活）／下背（穩）／胸椎（活）／肩胛（穩）
- 任何一個該活動的卡住 → 旁邊該穩定的代償
- 任何一個該穩定的太弱 → 旁邊該活動的犧牲

#### 1.3 動作系統三層（由被動到主動）
| 層級 | 英文 | 定義 |
|---|---|---|
| 柔軟度 | Flexibility | 被動產生動作的能力 |
| 活動度 | Mobility | 主動產生動作的能力 |
| 穩定度 | Stability | 強度下維持活動的能力 |

#### 1.4 影響活動度的 3 因素（診斷排查）
1. **肌肉張力**（拮抗肌太緊）→ 抑制伸展拮抗肌
2. **關節排列**（關節不在中心位）→ 關節鬆動術
3. **主動不足**（神經沒學會做）→ 訓練主動肌

#### 1.5 矯正訓練 4 步驟（核心 workflow）
```
INHIBIT → LENGTHEN → ACTIVATE → INTEGRATE
抑制    →  伸展    →  啟動    →  整合
↓         ↓          ↓          ↓
肌筋膜鬆 靜態/動態   大腦-肌肉   肌間協調
開緊繃   伸展       連結        動作次序
```

#### 1.6 姿勢進退階（由易到難）
躺姿 → 四足跪姿 → 跪姿 → 半跪姿 → 站姿 → 跨步
- 只想創造活動度 → 停在躺姿 + 四足跪
- 要用在訓練 → 至少走到半跪 + 站姿
- 要用在深蹲、跑步、變向 → 一定要跨步 + 單腳旋轉整合

### 二、呼吸與核心（地基）

#### 2.1 概念
- 良好呼吸模式 → 良好核心能力 → 良好動作模式
- 呼吸是一種動作：有主動肌、輔助肌、拮抗肌
- 呼吸與核心拮抗 = 產生穩定（寶特瓶/健身腰帶原理）
- 沒有最好的呼吸法，有問題的是「缺乏特定呼吸模式的能力」

#### 2.2 訓練：中立位保持腹內壓
1. 擺至中立位（注意頸椎，必要時墊枕）
2. 建立前側核心（鼻吸嘴吐，肋骨往內往下收）
3. 訓練後側擴張（保持前側張力，引導後側擴張）

### 三、下肢動作評估（Day 1）

#### 3.1 3 大動作測試（都看 3 個視角 · 每個 3 下）
| 動作 | 正面觀 | 背面觀 | 側面觀 |
|---|---|---|---|
| **負重深蹲** | 軀幹穩/膝對齊第二趾/骨盆不前後傾 | 軀幹穩/骨盆不旋轉不抬高 | 重心平均分配 |
| **硬舉** | 同上 | 同上 | 同上 |
| **分腿蹲** | 骨盆朝前/膝對齊第一到第二趾 | 軀幹穩/骨盆不旋轉不抬高 | 軀幹穩/骨盆不前後傾 |

#### 3.2 髖關節被動活動度參考（Passive ROM）
| 動作方向 | 標準角度 | 測量姿勢 |
|---|---|---|
| 屈曲 Flexion | 120° | 仰臥位，固定骨盆 |
| 伸展 Extension | 10° | 中立位向後 |
| 外展 Abduction | 45° | 大腿向外 |
| 內收 Adduction | 30° | 對側固定 |
| 外旋 External Rotation | 45-60° | 髖屈 90° |
| 內旋 Internal Rotation | 30-40° | 髖屈 90° |

#### 3.3 髖 6 方向 × 4 步驟矯正（矩陣）
每個方向都跑 抑制→伸展→啟動→整合：
- 髖屈曲：抑制髖後側 / 伸展腿後臀肌 / 啟動直膝抬腿 / 整合後跨弓箭步
- 髖伸直：抑制髖前側 / 伸展髂腰肌 / 啟動橋式 / 整合前跨弓箭步
- 髖外展：抑制髖內側 / 伸展青蛙式 / 啟動側躺外展 / 整合側跨
- 髖內收：抑制髖外側 / 伸展坐姿內收 / 啟動側躺內收 / 整合交叉跨
- 髖外旋：抑制髖前內 / 伸展鴿式 3 角度 / 啟動蚌殼式 / 整合轉體分腿蹲
- 髖內旋：抑制髖後外 / 伸展髖內轉 / 啟動側躺髖內轉 / 整合旋轉跨步

### 四、交叉測試拆解問題（核心技能）

**場景**：學員右腳膝蓋內扣

**可能的根本原因**（要交叉測試逐一排除）：
- 胸椎往右旋轉 → 骨盆代償
- 骨盆往左旋轉 → 右腿跟著轉
- 左髖屈受限 → 右側代償
- 左腳踝背屈受限 → 重心轉移
- 右足弓塌陷 → 內扣連鎖

**關鍵洞察**：膝蓋內扣 90% 不是膝蓋的問題。要往上下找源頭。

**排除邏輯範例（從 PDF）**：
- 把負重變前面，歪一邊就改善 → 問題在軀幹穩定
- 後腳跟墊槓片，歪一邊就改善 → 問題在腳踝背屈
- 右腳單腳穩，左腳骨盆不穩 → 問題在左側臀中肌

### 五、軀幹評估（Day 2）

#### 5.1 三面向評估
- **矢狀面**：坐姿屈伸（軀幹屈曲/伸展）
- **冠狀面**：坐姿側屈（軀幹側彎）
- **水平面**：坐姿旋轉（軀幹旋轉）

#### 5.2 軀幹 4 方向 × 4 步驟矯正
同樣跑 抑制→伸展→啟動→整合，針對屈/伸/側彎/旋轉

### 六、肩胛肱骨評估

- 矢狀面被動：肩胛前傾後傾 + 肱骨屈曲伸直
- 冠狀面被動：肩胛上下轉 + 肱骨內收外展
- 水平面被動：肩胛內外轉 + 肱骨水平內收外展
- 上肢動作測試：滑輪下拉（拉）+ 啞鈴胸推/斜板伏地（推）

### 七、課表編排邏輯

- 整合訓練動作設計範例：「右髖髖屈 + 軀幹右轉」這種跨平面整合
- 單邊 vs 雙邊？前負重 vs 背槓？要根據代償方向決定
- 從開放鏈 → 閉鎖鏈 → 整合

## 白話版（學員級 · 給 30+ 女性 TA 看）

### 一個最核心的洞察

> **身體的能力越全面，動作就越自由。**

站三七步、駝背圓肩、下樓梯傷膝蓋——這些都不是「姿勢問題」。
是身體的某個地方**能力不夠**，所以旁邊的地方跳出來扛。

這個「別人跳出來扛」，教練叫**代償**。
妳感受到的是——肩頸痠、腰痠、膝蓋卡、髖緊。

---

### 妳身體是這樣設計的（Joint by Joint）

從腳底往上數，妳的每一個關節都有**預設職業**：

| 關節 | 預設工作 |
|---|---|
| 腳踝 | 要會動（活動） |
| 膝蓋 | 要穩（穩定） |
| 髖 | 要會動（活動） |
| 下背 | 要穩（穩定） |
| 胸椎（上背） | 要會動（活動） |
| 肩胛 | 要穩（穩定） |

只要一個崗位缺工——旁邊的就會來救援。

比如腳踝不夠會彎，妳下蹲時膝蓋就扛；
比如胸椎不夠會轉，妳轉身時下背就扭。

長期下來：該動的越來越卡，該穩的越來越痠。
這就是妳 30+ 身體每天在發生的事。

---

### 身體動作的 3 個層級（Jamie 白話）

| 層級 | 白話解釋 | 例子 |
|---|---|---|
| 柔軟度 | 別人幫妳做，做得到 | 教練幫妳拉髖，髖可以到 120 度 |
| 活動度 | 妳自己做，做得到 | 妳自己抬腿，髖可以到 120 度 |
| 穩定度 | 有負重還能做到 | 背槓深蹲時髖還能到 120 度不變形 |

**最常見的問題：有柔軟度，但沒活動度**
→ 別人幫妳做可以，妳自己做卻做不出來
→ 因為大腦沒學會啟動那塊肌肉
→ 這就是「主動不足」

---

### 動作做不到 → 3 種可能（往下排查）

1. **肌肉張力**：拮抗肌太緊 → 要先鬆開它
2. **關節排列**：關節卡在錯的位置 → 要先校正
3. **主動不足**：大腦沒學會啟動 → 要練「神經連結」

這三個順序很重要——**鬆開 → 校正 → 啟動**，跳過任一步都沒用。

---

### 代償歸零 4 步驟（Jamie 進階版 3 步驟的根據）

原方法論 4 步：
1. **抑制**（用球/滾筒鬆開緊繃肌肉）
2. **伸展**（靜態/動態把那條肌肉拉長）
3. **啟動**（讓該醒的肌肉醒過來）
4. **整合**（把動作放回訓練裡）

Jamie 簡化 3 步：
1. **診斷**（知道代償在哪）
2. **先鬆緊的**（把抑制+伸展合併）
3. **重練基本動作**（把啟動+整合合併）

→ 給 30+ 女生 TA 用，3 步夠了

---

### 呼吸是地基

呼吸不好 → 核心不穩 → 動作全亂。

**怎麼練（3 步）**：
1. 躺好，頸椎中立（下巴微收，必要時墊枕頭）
2. 鼻吸嘴吐，吐氣時肋骨往內往下收（前側核心啟動）
3. 保持前側張力，再讓氣往後背擴張

→ 腰帶感覺 360 度都飽（不是只有肚子鼓）
→ 這樣一次呼吸，核心就上工一次

---

### 下蹲膝蓋內扣？不是膝蓋的問題

**真正要查的 5 個可能**：
1. 胸椎不夠轉 → 骨盆代償 → 膝蓋內扣
2. 骨盆歪掉 → 腿跟著轉 → 膝蓋跟著內
3. 對側髖屈不夠 → 這側補過來 → 內扣
4. 對側腳踝背屈不夠 → 重心偏移 → 內扣
5. 足弓塌陷 → 從底下一路連鎖上去

**要怎麼測？** 用排除法：
- 把重量放前面，內扣就改善 → 問題在軀幹
- 後腳跟墊個 2.5kg 槓片，內扣就改善 → 問題在腳踝
- 單腳蹲穩，雙腳蹲歪 → 問題在骨盆／臀中肌

---

### 妳該記住的一句話

> **30+ 女生身體的問題，9 成不是「練不夠」，是「練錯順序」。**

先診斷代償 → 鬆開緊的 → 啟動弱的 → 重練動作。
跳過任何一步，練越多代償越深。
