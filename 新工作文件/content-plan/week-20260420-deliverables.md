## 【YT Loom 第一支 · 5/10 · 「30+ 女生開始重訓的 5 個常見錯誤」】

### 結構（目標 10-12 分鐘）

**0:00-0:30 Hook（Talking head · webcam）**
「我帶課 12 年，從 2014 年的 World Gym 到現在自己的工作室。我觀察到一件事：30+ 女生開始重訓，大部分會犯一樣的 5 個錯。這 5 個錯我自己以前也犯過——今天我全部講一次，看完妳可以省 3-5 年摸索時間。」

**0:30-1:00 Agenda（簡報 page 1）**
今天講的 5 個錯誤：
1. 先瘦再練
2. 只做有氧不做肌力
3. 每次都做到力竭
4. 姿勢還沒對就加重
5. 把體重當唯一指標

**1:00-3:00 錯誤 1：先瘦再練（簡報 page 2-3）**

講稿：
```
第一個錯誤：先瘦再練。

我接到最多的諮詢就是：
「教練，我先瘦下來再開始重訓可以嗎？」
我都說⋯別別別。

30 歲以後的身體，先瘦再練是走冤枉路。
因為妳瘦下來的過程，如果沒有肌肉刺激
掉的很多是肌肉，不是脂肪。

結果呢？體重降了，但基礎代謝也降了。
一旦不再節食——反彈比之前還快。

正確順序是：先練肌力、再調飲食。
肌肉有了，身體會自己處理代謝。
不是反過來。
```

簡報：
- Page 2：「先瘦再練 = 掉肌肉不掉脂肪」圖示
- Page 3：順序對照：錯誤 vs 正確

**3:00-5:00 錯誤 2：只做有氧（簡報 page 4-5）**

講稿：
```
第二個錯誤：只做有氧不做肌力。

跑步 30 分鐘，燃燒的熱量 ≈ 200 大卡
吃一個便當就回來了。

真正長期改變體型的是肌肉量。
肌肉是妳體內最大的「代謝器官」。
肌肉量每增加 1kg，每天多燒 15-20 大卡——
一年下來就是 7000 大卡的差別。

有氧不是不做，是比例要改。
我建議：70% 肌力 / 30% 有氧。
大部分 30+ 女生的比例是反過來。
```

**5:00-7:00 錯誤 3：每次都做到力竭（簡報 page 6-7）**

講稿：
```
第三個錯誤：每次訓練都做到力竭。

這個錯誤來自「No pain no gain」的舊觀念。
但現代運動科學告訴我們⋯⋯
對 30+ 女生來說，這反而是反效果。

每次都做到力竭：
- 恢復時間拉長到 5-7 天
- 神經系統過度刺激
- 容易受傷
- 下次訓練品質下降

正確：留 1-2 下在「彈匣裡」（reps in reserve）
這樣妳可以一週練 3 次都有品質。
```

**7:00-9:00 錯誤 4：姿勢沒對就加重（簡報 page 8）**

講稿：
```
第四個錯誤：姿勢還沒對就加重。

這是 30+ 女生最容易受傷的地方。
妳看到別人舉 20 公斤

那個「身體記住了」的感覺⋯⋯
沒有教練在旁邊看
```
