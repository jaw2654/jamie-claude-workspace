## 段落 1 · 代償是什麼（0:45-3:30）

**主題**：弱肌/緊肌/神經斷 → 身體叫別的肌肉頂上 → 妳以為會了，其實練錯。

**3 要點照講**：
1. 代償 = 某肌肉太弱/太緊/神經連結斷，身體「叫別的肌肉頂上」完成表面動作
2. 妳看起來會了 → 其實是**用錯的方式做對的事**
3. 重訓越多，代償越深（因為妳一直在加強錯的版本）

**現場舉例**（挑 1）：
- 深蹲膝蓋內夾 → 臀中肌弱，大腿內側頂上
- 肩推聳肩 → 深層旋轉肌無力，斜方肌頂上
- 硬舉腰痠 → 髖後鏈弱，腰椎頂上

## 段落 2 · 為什麼妳每天都在加深（3:30-6:00）

**主題**：日常動作 + 錯的重訓 = 每天都在練壞。

**3 要點**：
1. 妳的身體每天做 5,000 次動作（走/坐/撿東西），代償也在這 5,000 次裡加強
2. 重訓時如果教練不修正，妳就是在「用更大重量練錯的版本」
3. 30+ 後神經連結會退化，代償會「記住自己」—— 越晚解，越難

**具體場景**：
- 辦公久坐 → 髖屈肌緊 → 走路用腰代償
- 產後腹直肌分離 → 核心斷線 → 深層肌不會用 → 一切動作都靠表面肌

→ 切 S4-S5

## 段落 3 · 怎麼解 · 代償歸零 3 步驟（6:00-9:00）

**主題**：先診斷，再喚醒，再重建動作模式。

**3 步驟（核心）**：
1. **診斷**：找出**妳在代償什麼**（不是全身檢查，是找 3-5 個關鍵弱點）
2. **喚醒**：讓斷線的肌肉先「會用」（從低重量/低負荷開始）
3. **重建**：動作模式重新建立（深蹲 / 硬舉 / 推 / 拉 四大動作檢查）

**現場講**：
> 很多人以為練 core / 練 glute 就夠
> 但妳如果不先做「診斷」——
> 妳會繼續練**錯的 core**、練**錯的 glute**。
> 代償歸零第一步，是認出來。
