# 報告格式規範（格式鎖死，除非 Jamie 主動改）

## 晨報（morning-briefing-daily）· Telegram 原生格式

### ⚠️ 格式已改（2026-04-15）——不再用 HTML 檔
- **推播**：Telegram `sendMessage` + `reply_markup` inline keyboard
- **不再**：sendDocument HTML 檔
- HTML 版本可備份存 `工作文件/每日簡報/` 但主體是 Telegram 訊息

### 結構（仿雷蒙晨報）

```
☁️ 早安，Jamie
<YYYY-MM-DD 星期X>

📅 今日行程
<HH:MM  活動 @ 地點>
<注意事項：時段重疊/交通>

✉️ 信件待辦
需要回覆的幾封信（優先處理）
- <sender> <主旨 + 重點>

✨ 今日 Top 3
1. <task>
2. <task>
3. <task>

<1 句提醒或行動指令>

動手吧。
```

按鈕：`[查看信件詳情] [今日完整行程] [收到]`（callback 或 URL）

### 內容來源
- 行程：Google Calendar
- 信件：Gmail `in:inbox is:unread newer_than:3d`（無則省略該段）
- Top 3：從昨天晚安回顧「明天接續」延續 + 今日行程 + 信件優先級推導
- 全球 / K-pop / 影片選題：改為**另一條獨立推播**（如果 Jamie 要保留）或移到週報
- ⚠️ 所有外部資訊必附真實 URL

## 週報（weekly-integrated-report）

- **模板**：`工作文件/週報/週報_template_TA深度版.html`
- **輸出**：`工作文件/週報/weekly-report-YYYY-MM-DD-v2.html`
- **推播**：Telegram sendDocument + 電腦 Safari 開
- **視覺**：背景 `#080808` · 強調 `#d8d8d8` · 黑白灰單色（CSS 變數 `--gold` 改此值）
- **Tab 機制**：JavaScript `function show(n)` + `onclick="show(n)"`
- **手機適配**：`@media max-width: 768px`

### 8 個 Tab

| Tab | 內容 |
|-----|------|
| 本週總覽 × TA | KPI、TA 輪廓、重大背景 |
| IG × TA 深度 | 觸及 / 非粉絲% / 各貼文表現 |
| YouTube 分析 | 觀看、訂閱、影片表現 |
| 韓系體態整合 | 韓國健身 / 時尚 / 藝人趨勢 |
| 學員動態 | 課堂數、反饋、新客 |
| 下週精準策略 | 基於數據的內容策略 |
| 週執行計劃 | 下週 IG / YT 發文計劃 |
| 首爾特輯 | 韓國/文化相關（或該週主題） |

## 交易日誌（trading-journal）

- **性質**：固定公開網站，所有歷史數據永久保留，每日累積不覆蓋
- **網址**：https://repurre-reports.pages.dev/交易日誌.html
- **本機**：`工作文件/交易日誌/交易日誌.html` → 部署到 `工作文件/public/`
- **平台**：Tradesea / Lucid Trading（TEST003/004，CME:NQ）
- **K 線圖**：Tradesea 截進出場箭頭，只截 K 線 · 存 `工作文件/public/nq-chart-YYYY-MM-DD.png`
- **手機適配**：sidebar 在手機隱藏

### ⚠️ 數據永久保留規則（不可流失）
- `EMBEDDED_TRADES` 陣列是**唯一真相來源**，只增不刪
- 每天盤後：抓進出場 → 新增到 EMBEDDED_TRADES 最前面 → 部署
- 盤前規劃：`localStorage.setItem('note_'+日期)` 存，並把當天筆記備份寫入 HTML
- K 線截圖按日期命名永久累積
- **禁止 localStorage.removeItem**
- **禁止覆蓋或刪除 EMBEDDED_TRADES 任何歷史交易**
- 帳戶爆倉換新帳號，舊數據保留不刪

### 交易日誌兩段式排程

| 排程 | 時間 | 自動做 |
|------|------|--------|
| 盤前 | 週一至五 15:00 | 趨勢初判 + 昨高/昨低/中值 + 重要數據 + 帳戶進度 → 部署 → Telegram 推 → Safari 開 |
| 盤後 | 週一至五 23:00 | 抓成交 + 計算損益（NQ $20/點）+ 進出場紀錄 + 數據實際值 + 帳戶餘額 → 部署 → 推 → 提醒 Jamie 填檢討 |


```
· <事項>

<時段 活動>
```
